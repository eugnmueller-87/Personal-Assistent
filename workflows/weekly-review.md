# Weekly Review

**Trigger:** Every Monday morning  
**Owner:** Human (to be automated)  
**Output:** Updated TODO.md, closed issues, plan for the week

## Steps
1. Review all open GitHub Issues
2. Update TODO.md — move done items, reprioritize
3. Check ROADMAP.md — are we on track?
4. Review each active project in `projects/`
5. Set the top 3 priorities for the week
6. Optional: generate AI summary (Claude agent)

## Automations
- [ ] GitHub Actions: auto-create a weekly review issue every Monday
- [ ] Claude agent: summarize open todos and post to phone via push notification
